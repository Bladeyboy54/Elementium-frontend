### Installation

   #### Front-end Installation
1. Clone the frontend repo
   ```sh
   https://github.com/Bladeyboy54/Elementium-frontend.git
   ```
2. Install the node modules for Electron
   ```sh
   cd elementium
   ```
   then
   ```sh
   npm i
   ```
3. Install the node modules for React 
   ```sh
   cd elementium-app
   ```
   then
   ```sh
   npm i
   ```
